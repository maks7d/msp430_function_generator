# msp430_function_generator

Downloads : 
- https://www.ti.com/tool/MSP430-FLASHER#downloads
- https://www.ti.com/tool/download/MSP430-GCC-OPENSOURCE/9.3.1.2

Make a folder named "ti" and put the downloaded files in it.

`sudo make flash` to flash the microcontroller. You can also use `make clean` to remove generated files.
